# lem-in

***runs on macOS only***

![visualizer gif](https://nextcloud.42l.fr/s/Y8G82ddsBJsZN3H/download/ezgif-5-3b145d556f.gif)

## compile
`make`

## launch program
`./lem-in < maps/subject/subject3.map`

## help
`./lem-in -h`

## compile bonus
`make bonus`

## visualize ant farm (bonus only)
`./lem-in_bonus -v < maps/subject/subject3.map`
